# Wanderlust – Travel Booking Web Application

A full-stack MERN-based travel booking web application with authentication, booking management, RESTful APIs, and responsive UI.

---

## Features

- User Authentication & Authorization
- Create, Edit, Delete Listings
- Booking Functionality
- RESTful APIs
- Responsive Design
- MongoDB Integration
- API Testing with Postman

---
